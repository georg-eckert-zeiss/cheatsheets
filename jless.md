# jless

jless ([jless.io](https://www.jless.io)) is a fast and versatile JSON viewer for the command line.

|Hotkey|Description|
|-|-|
|`{right}`|expand|
|`{right}`|collapse|
|`e`|expand all siblings|
|`c`|collapse all siblings|
|`/<pattern>`|**find** "pattern"|
|**find** > `n`|find next|
|**find** > `N`|find previous|
